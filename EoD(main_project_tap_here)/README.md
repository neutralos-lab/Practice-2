# Notae Service

## Описание
Сервис шифрования данных с использованием методов Виженера и Цезаря.

## Установка и запуск

Скачать файлы (папка EoD)

Открыть папку EoD в среде разработки

Установить flask

Запустить файл EoD_Service.py

Перейти на локальный сервер
